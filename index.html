Ekspresi Dasai OS

Lebih dari 50 efek animasi yang bereaksi terhadap sentuhan Anda. Ciptakan pengalaman dasbor hidup dengan kecepatan dan jeda yang disesuaikan.

📱

Sinkronisasi Chronos

Terhubung dengan HP via Bluetooth (Aplikasi Chronos). Dapatkan info Cuaca dan sinkronisasi real-time kapanpun.

👆

Menu 3-Tap Interaktif

Sentuh 3 kali untuk masuk ke menu rahasia. Anda dapat mengatur Bluetooth, layar Invert Color, dan Sleep Mode seketika.

🗺️

Petunjuk Arah Maps

Menampilkan panah arah, jarak, dan estimasi waktu langsung di layar OLED Anda saat menggunakan navigasi Google Maps di HP.

⏰

Tampilan Jam Pintar

Jam presisi tinggi yang terkalibrasi dengan HP Anda. Selalu update dan sinkron meski Anda sedang mematikan Bluetooth.

💬

Notifikasi Panggilan & Aplikasi

Menampilkan ID penelepon, pesan WhatsApp, Instagram, dan notifikasi aplikasi lainnya langsung di layar mungil Anda.

Tutorial Mengonfigurasi Dasai

Dasai Custom OS memiliki fitur Web Serial. Anda tidak perlu membongkar hardware hanya untuk mengganti pin.

  1. Buka web konfigurasi DASAI.
  2. Pilih PIN yang ingin anda ubah
  3. Klik Save Config dan Reboot untuk merestart ESP32 agar perubahan diterapkan.
  4. Pasang aplikasi Chronos di Android untuk melakukan pair Bluetooth agar mendapat notifikasi & cuaca.
